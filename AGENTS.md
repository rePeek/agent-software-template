# Agent Software Template

This repository is a generic template for building, operating, and maintaining agent-oriented software systems.

## Project Goal

Provide a neutral starting point for agent projects that need clear execution rules and maintainable agent operating assets.

## Project Architecture

`AGENTS.md` is the repository-level entry point for agents.

All project operating logic is defined and maintained in `.agent/ARCHITECTURE.md`.

Agents must use `.agent/ARCHITECTURE.md` when executing user commands, including process selection, skill usage, rule application, registry updates, and operating-asset maintenance.
